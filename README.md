# WhatsApp Bot com Python + Flask

Bot de atendimento automático para WhatsApp, desenvolvido utilizando a **WhatsApp Cloud API (Meta)** e **Flask**.

##  Funcionalidades
 Resposta automática a mensagens
 Menu interativo por opções numéricas
 Integração oficial com a API do WhatsApp
 Estrutura simples e escalável

## Tecnologias
Python
Flask
WhatsApp Cloud API
Ngrok (ambiente de testes)- VS Code

## Objetivo
Projeto criado com foco em aprendizado, automação de atendimento e construção de portfólio na área de desenvolvimento e computação.

## Observações
Este projeto utiliza a API oficial do WhatsApp, seguindo as diretrizes da Meta.
